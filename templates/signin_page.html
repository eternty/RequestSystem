EterntyServiceDesk

Автоматизированная система службы поддержки
информационно-вычислительной сети клиента

Система предназначена для эффективного управления запросами клиентов службы поддержки.

Система позволяет:
  • регистрировать обращения (заявки) с разным приоритетом,;
  • назначать исполнителя для проведения работ по заявкам;
  • отслеживать выполнение работ по заявкам;
  • взаимодействовать заказчикам с исполнителями в рамках решения проблемы.
